Welcome to r4atlantis, home for code developed by the Atlantis community

r4atlantis is meant to be your link to existing code for developing, executing, and comparing Atlantis models in R. We will provide forked copies of existing projects that are regularly updated, and we hope this space can also serve as a common development area where new packages can be designed (and existing ones merged).

If you have code you'd like to contribute or would like to join the group or a specific team, please contact @jsgosnell or @thefaylab
